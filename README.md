# Class Timetable Generator

A web-based timetable generator developed using Node.js and MySQL.

## Features
- Admin dashboard
- Add timetable entries
- Update timetable
- Delete timetable
- Student timetable view

## Technologies Used
- HTML
- CSS
- JavaScript
- Node.js
- MySQL

## Database Setup
Import the database file:
class_timetable_db.sql

## How to Run
1. Install Node.js
2. Import database in MySQL
3. Run server:
   node server/app.js
4. Open the client pages in browser
